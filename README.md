# Natively

Natively is a local-first proactive meeting and workflow companion for desktop use.

## What It Does

- Watches live screen and meeting context
- Helps you stay accurate, on-message, and unblocked in real time
- Stores meeting memory locally
- Supports BYOK and local model workflows
- Packages as a native Electron desktop app for Windows and macOS

## Local Development

```bash
npm install
npm run app:dev
```

## Build

```bash
npm run build
npm run app:build
```

## Notes

- Auto updates are disabled by default unless explicitly enabled through environment configuration.
- This repository is intentionally stripped of legacy marketing, creator branding, and old use-case framing.
