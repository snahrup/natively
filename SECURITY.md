# Security Policy

Natively is a local-first desktop application. Security issues should be treated seriously.

## Reporting

If you discover a security issue in the codebase or packaged application, report it to the maintainer or distributor of the local repository you are using.

## Scope

- Prompt injection or data exposure issues
- Local credential handling issues
- Improper persistence or logging of private meeting data
- Update or packaging integrity issues
