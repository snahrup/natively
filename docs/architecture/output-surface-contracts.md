# Output Surface Contracts

## Meeting Coach

Audience: user speaking live in a meeting

Format:

- `SAY THIS`
- `CORRECTION`
- `DATA`
- `HEADS UP`

Rules:

- first person
- speakable immediately
- short

## Ambient Coach

Audience: user outside a meeting

Rules:

- direct-to-user
- minimal
- reminder or correction oriented

## Meeting Prep

Audience: user before a meeting

Sections:

- summary
- ranked context bullets
- related meetings
- commitments
- checklist
- open questions

Rules:

- deterministic
- scannable
- evidence-first

## Reactive Q&A

Audience: user explicitly asked a question

Rules:

- concise
- uncertainty when needed
- provenance available even if not shown verbatim
